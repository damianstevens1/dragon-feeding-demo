## eat-crunch.wav

- Source: OpenGameArt, "Apple Bite" by AntumDeluge
- URL: https://opengameart.org/content/apple-bite
- Downloaded source file: `apple_bite.ogg`
- Local file: `eat-crunch.wav`
- License: CC0 / public domain
- Conversion: OGG/Vorbis converted to mono 44.1 kHz WAV for local app playback.
- Edit: trimmed to the first bite segment, about 0.223s, so the cue aligns with the mascot's quick eat animation.
