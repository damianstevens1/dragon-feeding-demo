# App Store Assets

This folder has the App Store Connect prep material for Dragon Feeding Game.

## Included

- `app-store-connect.md`: app record, pricing, category, privacy, and listing copy
- `submission-checklist.md`: remaining steps for Xcode and App Store Connect
- `review-notes.md`: text for Apple review notes
- `screenshots/iphone-6.9/`: draft 1290 x 2796 App Store screenshots
- `screenshots/source/`: smaller preview copies

## Regenerate Screenshot Drafts

From the repo root:

```bash
pnpm appstore:screenshots
```

These are polished draft screenshots made from the same local app assets. Once full Xcode is installed, simulator screenshots can replace them if desired.
