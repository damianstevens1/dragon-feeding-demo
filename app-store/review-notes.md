# App Review Notes

Use this text in App Store Connect review notes.

```text
Dragon Feeding Game is a child-friendly educational flashcard game. It does not require login, does not collect user data, and does not contain ads or purchases.

To test:
1. Tap Play.
2. Choose a practice mode if desired.
3. Tap or drag a snack to the dragon.
4. Listen to the question.
5. Choose an answer.
6. Continue through the cards to see the reward video.

Audio starts only after user interaction, which is required for iPhone playback behavior.
```
