# Submission Checklist

## Required On The Mac

- Full Xcode 26 or newer installed
- Apple Developer Program membership
- Node.js installed
- `pnpm` installed
- This repository pulled locally
- Current App Store screenshots are in `app-store/screenshots/iphone-6.9`

Install `pnpm` if needed:

```bash
npm install -g pnpm
```

## Required In Xcode

- Open `ios/DragonFeedingGame.xcodeproj`
- Select the Apple Developer team
- Confirm bundle ID
- Confirm version `1.0`
- Confirm build number `1`
- Archive using **Product > Archive**
- Upload to App Store Connect from Organizer

## Required In App Store Connect

- Create iOS app record
- Use matching bundle ID
- Add screenshots
- Fill app description, keywords, support URL, and review notes
- Set privacy to **Data Not Collected**
- Set pricing to **Free**
- Submit for review

## Cannot Be Completed By Codex Without Your Apple Account

- Apple Developer team selection
- Certificate/provisioning/signing approval
- Bundle ID registration if Apple requires it
- App Store Connect app creation
- Pricing confirmation
- Final submit for review
